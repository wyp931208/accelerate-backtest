# XPK加速策略 用户手册

## 一、策略概述

XPK加速策略是一种基于量价特征的短线交易策略，专注于捕捉创业板股票放量加速上涨后的短线机会。策略通过多重条件筛选信号，在信号日收盘买入，设置止盈目标与补仓机制，在规定持有期内择机卖出。

---

## 二、信号筛选条件

信号需**同时满足**以下所有条件：

### 2.1 板块与基础过滤

| 条件 | 说明 |
|------|------|
| 仅创业板 | 股票代码以 `300` 开头 |
| 非ST股 | 股票名称不含"ST"字样 |
| 非次新股 | 上市日期距交易日 ≥ 60个自然日 |
| 非停牌 | 当日未停牌 |

### 2.2 涨跌幅条件

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 涨跌幅下限 | 2.0% | 当日涨跌幅 ≥ 此值 |
| 涨跌幅上限 | 8.0% | 当日涨跌幅 ≤ 此值 |

**计算方式**：`涨跌幅 = (收盘价 - 前收盘价) / 前收盘价 × 100%`，即 Tushare 提供的 `pct_chg` 字段。

### 2.3 量比条件

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 量比下限 | 1.5 | 当日量比 ≥ 此值 |
| 量比上限 | 2.5 | 当日量比 ≤ 此值 |

**计算方式**：
- 优先使用 Tushare `daily_basic` 接口提供的 `volume_ratio` 字段
- 若该字段缺失（NaN），则回退计算：`量比 = 当日成交量 / 前一交易日成交量`
- 注意：此回退计算为简化版量比（标准量比应为当日成交量 / 过去5日平均成交量）

### 2.4 上影线比例条件

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 上影线比例下限 | 0.25 | 上影线比例 ≥ 此值 |
| 上影线比例上限 | 0.50 | 上影线比例 ≤ 此值 |

**计算方式**：

```
上影线比例 = (最高价 - 收盘价) / (最高价 - 最低价)
```

> 当最高价 = 最低价（一字涨停/跌停）时，上影线比例记为 0。
> 上影线比例越大，说明收盘价离当日最高价越远，多方冲高回落的力度越大。

### 2.5 VWAP 条件

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 要求收盘高于VWAP | 是 | 勾选后要求收盘价 ≥ VWAP × (1 + 百分比) |
| 收盘高于VWAP百分比 | 0.3% | 收盘价需高于VWAP的百分比 |

**VWAP计算方式**：

```
VWAP = 成交额 / 成交量
```

> VWAP（Volume Weighted Average Price）即成交量加权平均价，代表当日资金的真实平均成本。

**筛选逻辑**：
- 若勾选且百分比 > 0：`收盘价 ≥ VWAP × (1 + 百分比/100)`
- 若勾选且百分比 = 0：`收盘价 ≥ VWAP`
- 若不勾选：跳过此条件

### 2.6 前N日累计涨幅条件

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 前N日(N) | 20 | 回溯天数 |
| 累计涨幅下限 | 20.0% | 前N日累计涨幅 > 此值 |
| 累计涨幅上限 | 100.0% | 前N日累计涨幅 ≤ 此值 |

**计算方式**：

```
前N日累计涨幅 = 前20个交易日的涨跌幅(pct_chg)之和
```

> 使用滚动窗口（rolling window）计算，需要完整的N个交易日数据才计算，否则记为 NaN。

## 三、交易规则

### 3.1 买入规则

| 规则 | 说明 |
|------|------|
| 买入时机 | 信号日当日收盘价买入 |
| 买入金额 | 按设定的单次买入金额计算股数 |
| 股数计算 | `股数 = 买入金额 / 收盘价 / 100 × 100`（按100股取整） |
| 涨停不买 | 若信号日涨停（涨幅 ≥ 9.8%主板 / 19.8%创业板，且收盘价 ≥ 涨停价×0.995），则跳过该信号 |
| 停牌不买 | 若信号日停牌，则跳过该信号 |

**涨停价计算**：
- 主板：`涨停价 = 前收盘价 × 1.10`（四舍五入到分）
- 创业板：`涨停价 = 前收盘价 × 1.20`（四舍五入到分）

### 3.2 补仓规则

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 补仓比例 | 0.86 | 股价跌破 `初始买入价 × 补仓比例` 时触发补仓 |

**补仓逻辑**：
1. 触发条件：持有期内某日最低价 ≤ `买入均价 × 补仓比例`
2. 补仓价格 = `买入均价 × 补仓比例`
3. 补仓股数 = 与初始买入股数相同（按100股取整）
4. 补仓后成本 = `(初始总投入 + 补仓投入) / 总股数`
5. 补仓仅触发一次，不重复补仓
6. **补仓后持有期重置**：补仓日重新计算最大持有天数

**补仓后成本示例**：
```
初始买入价 = 10.00元，买入 1000 股，投入 10,000元
补仓比例 = 0.86，补仓价 = 10.00 × 0.86 = 8.60元
补仓 1000 股，投入 8,600元
补仓后成本 = (10,000 + 8,600) / 2,000 = 9.30元/股
```

### 3.3 卖出规则

系统会遍历 0.5%、1%、2%、...、20% 共21个止盈目标，分别计算每个目标下的交易结果。

**卖出优先级**：

#### 情况一：未补仓

| 卖出方式 | 条件 | 卖出价格 |
|----------|------|----------|
| 止盈卖出 | 某日最高价 ≥ `买入价 × (1 + 止盈目标/100)` | `min(当日最高价, 目标价)` |
| 持有到期未达标 | 到达最大持有天数最后一日仍未止盈 | 当日收盘价 |
| 数据截止强制卖出 | 持有期内数据不足（到达回测期末） | 最后一日收盘价 |

> 未补仓时，买入次日（T+1）起即可判断止盈卖出。

#### 情况二：已补仓

| 卖出方式 | 条件 | 卖出价格 |
|----------|------|----------|
| 止盈卖出 | 某日最高价 ≥ `补仓后成本 × (1 + 止盈目标/100)` | `min(当日最高价, 目标价)` |
| 补仓后持有到期卖出 | 补仓后重置的最大持有天数最后一日仍未止盈 | 当日收盘价 |
| 数据截止强制卖出 | 持有期内数据不足 | 最后一日收盘价 |

> 已补仓时，补仓次日（补仓日+1）起才可判断止盈卖出。补仓日当天不判断卖出。

### 3.4 收益计算

```
投入金额 = 初始买入金额 + 补仓金额（如有）
卖出收入 = 总股数 × 卖出价格
盈利金额 = 卖出收入 - 投入金额
收益率 = 盈利金额 / 投入金额
```

## 四、回测汇总指标说明

### 4.1 按板块 × 止盈点汇总

| 指标 | 计算方式 |
|------|----------|
| 总交易数 | 该板块在该止盈点下的交易笔数 |
| 胜率 | 盈利交易数 / 总交易数 |
| 平均交易日数 | 所有交易的平均持有交易日数 |
| 单位时间收益率 | 整体收益率 / 平均交易日数 |
| 总投入 | 所有交易的投入金额之和 |
| 总盈利 | 所有交易的盈利金额之和 |
| 整体收益率 | 总盈利 / 总投入 |

> **单位时间收益率**是衡量策略效率的核心指标，值越高说明资金利用率越好。

## 五、复权方式说明

| 复权方式 | 说明 | 适用场景 |
|----------|------|----------|
| 前复权（qfq） | 以最新价格为基准，向前调整历史价格 | **推荐**。保持最新价格不变，历史价格向下调整，便于观察真实涨跌 |
| 后复权（hfq） | 以上市首日价格为基准，向后调整价格 | 观察长期真实收益，价格可能远高于当前市价 |
| 不复权（none） | 使用原始交易价格 | 观察历史真实成交价格 |

**计算方式**：
- 前复权：`调整后价格 = 原始价格 × 当日复权因子 / 最新复权因子`
- 后复权：`调整后价格 = 原始价格 × 当日复权因子 / 最早复权因子`

> 注意：无论选择何种复权方式，涨停判断始终使用原始价格（除权价格），因为涨跌幅限制基于除权价格计算。

## 六、K线图技术指标说明

### 6.1 MACD（移动平均收敛/发散指标）

| 指标 | 参数 | 计算 |
|------|------|------|
| DIF（快线） | EMA(12) - EMA(26) | 12日指数移动平均 - 26日指数移动平均 |
| DEA（慢线） | EMA(DIF, 9) | DIF的9日指数移动平均 |
| MACD柱 | 2 × (DIF - DEA) | DIF与DEA差值的2倍 |

### 6.2 PSY（心理线指标）

```
PSY = N日内上涨天数 / N × 100（N=12）
```

> PSY > 75 表示超买，PSY < 25 表示超卖。

### 6.3 MA（移动平均线）

默认显示 MA5、MA10、MA20、MA30。

### 6.4 成交量MA

默认显示 5日和10日成交量移动平均。

## 七、参数调优建议

如果回测显示"未产生有效交易"，可按以下顺序调整参数：

1. **降低量比下限**：从1.5降到1.0，或扩大量比上限到5.0
2. **降低累计涨幅下限**：从20%降到10%或5%
3. **降低上影线比例下限**：从0.25降到0.10
4. **取消VWAP条件**：不勾选"要求收盘高于VWAP"
5. **扩大日期范围**：覆盖更多交易日，增加信号数量

回测结果中会显示**信号筛选诊断**，逐条件显示每步过滤后的剩余数量，帮助定位哪个条件过滤过严。
